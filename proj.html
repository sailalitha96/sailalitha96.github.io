Lung Nodule Detection( LUNA)

Developed a classification algorithm for Lung Nodule Analysis. Data preprocessing and augmentation on 3d CT scans and implemented segmentation techniques for optimizing performance. Implementing Deep learning Networks- Squeeze Net, Mobile Net (depth wise spatial convolution) with traditional 2D convolutional network to compare the performance. Overview of the Project

Autonomous Beach Cleaning Robot

Developed an autonomous beach cleaning robot to automate litter picking in coastal regions. I was involved in design process. Machine learning for litter detection using custom CNN layers in Tensorflow were implemented real-time. Video of the robot working . The project was in colloboration with Sridhar Thiagarajan.This project was fully funded under the innovation centre for research -Undergraduates
